# propotyperc.h
A simple, unofficial C API for [Propotype RC](https://propotyperc.com/) cars

## Quick Start
Copy [propotyperc.h](https://github.com/noadhum/propotyperc.h/blob/master/propotyperc.h) into your project and include it:
```c
#define PROPOTYPERC_IMPLEMENTATION
#include "propotyperc.h"
```
